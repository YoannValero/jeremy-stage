# Stage Jérémy

## Jeremy's challenge

## Objectifs :

- Création d'un site internet de présentation de la Préfecture des Pyrénées-Atlantiques
- Présenter brièvement les différents métiers observés
- Initiation à HTML et CSS


## Structure du site

Le site doit etre construit de la façon suivante avec les 3 fichiers et 2 dossiers suivants :

![structure_site](https://github.com/YoannValero/jeremy-stage/blob/main/structure%20site.png)

- Le dossier files devra contenir les fiches des métiers observés.
- Le dossier img devra contenir les images du site.
- index.html sera la page d'accueil du site qui contiendra la présentation de la Préfecture
- index.css devra contenir les règles de style (Couleurs, positionnement des éléments de la page etc...)
- metiers.html sera la page présentant les métiers observés durant le stage

### Éditeur de code 

- Utiliser Visual studio Code (déjà installé sur le PC)

Après avoir créer les dossiers et fichiers, il suffira d'ouvrir le dossier dans Visual Studio code.


### Exécution du code dans Visual Studio Code

Ouvrir le dossier puis lancer le serveur de développement
![GoLive](https://github.com/YoannValero/jeremy-stage/blob/main/golive.png)
